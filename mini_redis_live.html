사용 포인트

  • 이 페이지는 ws://.../ws 로 직접 연결됩니다.
  • benchmark는 실제 source DB 조회와 Mini Redis cache 조회를 모두 실행합니다.
  • cache miss 시 source DB에서 읽고 Mini Redis에 저장한 뒤, 이후 요청은 cache hit로 처리합니다.
